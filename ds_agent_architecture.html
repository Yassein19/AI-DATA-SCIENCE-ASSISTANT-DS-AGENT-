DATA FLOW
USER  → uploads file / enters SQL query  → INGESTION LAYER  → parsed into pd.DataFrame
    → PROCESSING LAYER  → df_summary() generates text profile
    → AI LAYER  → prompt + profile sent to claude-sonnet-4 via HTTPS
    → Claude response returned as markdown string
    → VIZ LAYER  → Plotly charts rendered alongside AI insights
    → OUTPUT LAYER  → user sees charts + AI narrative + download button
